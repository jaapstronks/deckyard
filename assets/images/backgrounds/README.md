# Demo backgrounds

The `demo-*.jpg` images in this folder are abstract gradient renders
generated programmatically for this project (SVG radial gradients rasterized
with sharp, plus film grain). They are original works with no third-party
source material and are released under the same MIT license as the rest of
the repository — use them freely.

They exist as tasteful defaults for background-image presets
(`theme.backgroundPresets`), gallery/sample content, and as the AI pipeline's
placeholder image. Themes are expected to point `backgroundPresets` at their
own imagery; see `docs/developer/themes.md`.
