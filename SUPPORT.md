# Support

Need help with Deckyard? Pick the channel that fits:

- **Questions, ideas, "how do I…"** → open a
  [Discussion](https://github.com/jaapstronks/deckyard/discussions). This keeps
  the issue tracker focused on confirmed bugs and concrete work.
- **Bugs & feature requests** → open an
  [issue](https://github.com/jaapstronks/deckyard/issues/new/choose) using the
  templates.
- **Security vulnerabilities** → do **not** open a public issue. Use private
  reporting via the repository's Security tab. See [`SECURITY.md`](SECURITY.md).
- **Setup & self-hosting** → start with the [`README.md`](README.md) and the
  [`docs/`](docs/) folder.

Support is provided on a best-effort basis by a single maintainer. See
[`GOVERNANCE.md`](GOVERNANCE.md) for what to expect.
